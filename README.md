## Installation

Ensure you have the correct version of Node.js and npm installed:

Recommended:

- Node: `v16.4.0`
- npm: `v7.18.1`

Minimum:

- Node: `v14.17.1`
- npm: `v7.18.1`

Open a terminal with Node.js installed
```
npm i honk.js@latest --save
```
